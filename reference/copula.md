# Copula

There are three copula series in Kaitag language. Let's consider them by their purpose.

## 1. Facts & States

| :---------: | :-------: | :-----------: | :-----: | :-------------------: |
| | --I, we-- | --you (sg.)-- | --you-- | --he, she, it, they-- |
| --present-- | ==–¥–∞== | ==–¥–µ== | ==–¥–∞== | ==—Ü–µ== |
| --past-- | ==–¥–µ== ||||
{.bl}

The basic series corresponds to _"to be"_ and has two usages:

1. Denoting _facts_:

   - ==–¥—É –ú—è–≥—å—è–º–∞–¥ –¥–∞==  
     --I am Muhammad;--
   - ==–∏ —Ç—É—Ö—Ç—É—Ä –¥–µ==  
     --you are a doctor;--
   - == –¥–∏–ª–∞ –≥–æ–∑–µ —Ü–µ==  
     --[that is] my honey.--

2. Conveying _state_:
   - ==–¥—É —É–ª—á”è—É–Ω –¥–∞==  
     --I am studying;--
   - ==–Ω–∏—à–∞ –∫—å–∞–Ω–ª–µ –¥–µ==  
     --you were late;--
   - ==–¥–∞–º –≥—É–Ω–∞–ª —Ü–µ==  
     --I'm warm (lit. it's warm to me).--

::: warning
The _2nd person singular_ is the same as the _past_.
:::

::: tip
Remember that the [adverbial forms](./adjective-adverb) convey state.
:::

### False

This copula has irregular negative counterparts:

| :-----: | :----: | :-------: | :-: | :---------------: |
| | --I, we-- | --you (sg.)-- | --you-- | --he, she, it, they-- |
| --present-- | ==–∞–∫–∫–æ–¥== | ==–∞–∫–∫–æ—Ç—Ç–µ== | ==–∞–∫–∫–æ—Ç—Ç–∞== | ==–∞–∫–∫—É== |
| --past-- | ==–∞—á—á—É== ||||
{.bl}

They are used to deny facts:

- ==–¥—É —Ç—É—Ö—Ç—É—Ä –∞–∫–∫–æ–¥;==  
  --I am not a doctor.--
- ==–≥—å–µ–ª –≥–æ–∑–µ –¥–∏–ª–∞ –∞–∫–∫—É;==  
  --that honey is not mine.--
- ==–Ω–∏—à–∞ –∫—å–∞–Ω–ª–µ –∞—á—á—É;==  
  --you were not late.--

::: details üìú History
The past form's consonant alternation is due to the historic process:

1. ==–∞–∫–∫–≤–∏==, where ==–∞–∫–∫–≤== is the root and ==-–∏== is [past habitual](./verb);
2. ==–∞—á—á–≤–∏== ‚Äî palatalization;
3. ==–∞—á—á—É== ‚Äî delabialization.

:::

## 2. Presence

| :-----: | :----: | :-------: | :-: | :---------------: |
| | --I, we-- | --you (sg.)-- | --you-- | --he, she, it, they-- |
| --present-- | ==–±–∏ –¥–∞== | ==–±–∏ –¥–µ== | ==–±–∏ –¥–∞== | ==–±–∏== |
| --past-- | ==–±–∏ –¥–µ== ||||
{.bl}

::: warning
The copula has class agreement: ==–±–∏==, ==–≤–∏==, ==—Ä–∏==, ==–¥–∏==. For simplicity the table provides them in the neuter class, but know that e.g. the 1st & 2nd person plurals are _always_ ==–¥–∏==.
:::

The second series is used for presence:

- ==–¥—É —Ö—ä–∏–ª–∏ –≤–∏ –¥–∞==  
  --I am [currently] at home;--
- ==—É–±–∞ –¥—É—Ü—Ü–∏ —Ä–∏==  
  --mother is with me;--
- ==–∏–ª–∞ –≥–æ–∑–µ –±–∏ –¥–µ==  
  --you had honey.--

::: tip
The person agreement is marked by appending the corresponding _"to be"_ forms (except for the 3rd person where it's zero). The basic copula is used this way with the other copulas too as we shall see.
:::

### Absence

The irregular negative counterparts are:

| :-----: | :----: | :-------: | :-: | :---------------: |
| | --I, we-- | --you (sg.)-- | --you-- | --he, she, it, they-- |
| --present-- | ==–±–µ–π–∫–∫–æ–¥== | ==–±–µ–π–∫–∫–æ—Ç—Ç–µ== | ==–±–µ–π–∫–∫–æ—Ç—Ç–∞== | ==–±–µ–π–∫–∫—É== |
| --past-- | ==–±–µ–π—á—á—É== ||||
{.bl}

::: warning
This copula too has class agreement: ==–±–µ–π–∫–∫—É==, ==–≤–µ–π–∫–∫—É==, ==—Ä–µ–π–∫–∫—É==, ==–¥–µ–π–∫–∫—É==.
:::

These forms convey absence:

- ==–¥—É —Ö—ä–∏–ª–∏ –≤–µ–π–∫–∫–æ–¥==  
  --I am not at home;--
- ==–∏–ª–∞ –≥–æ–∑–µ –±–µ–π–∫–∫—É==  
  --you don't have honey.--

::: details üìú History
Historically this is fusion of the positive forms with the negative forms of _"to be"_:

- ==–±–µ–π–∫–∫–æ—Ç—Ç–µ== ‚¨ÖÔ∏è ==–±–∏== + ==–∞–∫–∫–æ—Ç—Ç–µ==;
- ==–±–µ–π—á—á—É== ‚¨ÖÔ∏è ==–±–∏== + ==–∞—á—á—É==.

:::

::: details ü§î Experimental
It seems that speakers have no trouble understanding simplified negative forms made via regular verbal negation prefix ==a-== (see the next copula):

- ==–≤–µ–π–∫–∫—É –¥–∞== ‚ùì ==–∞–≤–∏ –¥–∞==;
- ==–≤–µ–π—á—á—É== ‚ùì ==–∞–≤–∏ –¥–µ==.

Though it makes them laugh. üòÅ
:::

## 3. Availability

| :---------: | :--------: | :-----------: | :--------: | :-------------------: |
| | --I, we-- | --you (sg.)-- | --you-- | --he, she, it, they-- |
| --present-- | ==–±–µ–ª –¥–∞== | ==–±–µ–ª –¥–µ== | ==–±–µ–ª –¥–∞== | ==–±–µ–ª== |
| --past-- | ==–±–µ–ª –¥–µ== ||||
{.bl}

::: warning
Just like the previous two, this one too has class agreement: ==–±–µ–ª==, ==–≤–µ–ª==, ==—Ä–µ–ª==, ==–¥–µ–ª==.
:::

The third series roughly means "to have been":

- ==–¥—É —Ö—ä–∏–ª–∏ —Ä–µ–ª –¥–∞==  
  --I am still at work;--
- ==—É—Ü—Ü–∏ —É–ª—á”è—É–Ω –≤–µ–ª==  
  --brother has been studyin.--

Fortunately it is negated via the regular verbal prefix ==–∞-==:

- ==—É—Ü—Ü–∏ —É–ª—á”è—É–Ω –∞–≤–µ–ª==  
  --brother is no longer studying;--
- ==—Å–æ –≥–æ–∑–µ –∞–±–µ–ª –¥–µ==  
  --yesterday there was no honey [left].--

## Summary

The three copula series are used for:

1.  ==—Ü–µ== & ==–∞–∫–∫—É== ‚Äî facts:
    - ==–∏–ª–∞ –≥–æ–∑–µ —Ü–µ==  
      --[this] is your honey;--
2.  ==–±–∏== & ==–±–µ–π–∫–∫—É== ‚Äî presence:
    - ==–∏–ª–∞ –≥–æ–∑–µ –±–∏==  
      --you have honey;--
3.  ==–±–µ–ª== & ==–∞–±–µ–ª== ‚Äî availability (continious presence):
    - ==–∏–ª–∞ –≥–æ–∑–µ –±–µ–ª==  
      --you still have honey (lit. your honey remains).--

They agree in person and time, and two of them have noun classes. Here are interactive examples in which they control [verbal participles](./verb):

  | -------------- | ----- | --------- | -------- |
  |                | –¥—É–ª   | –∏–ª        | –≥—å–µ–ª—Ç—Ç–∞–ª |
  |                |
  | –±—É—Ä—Å–∏–ª         |
  |                |
  |                | –¥–∞    | –¥–µ        | —Ü–µ       |
  | <br>           |       |           |          |
  |                | --I-- | --you--   | --they-- |
  |                |
  | --have told;-- |

  {.context}

- |              | present | past    |
  | ------------ | ------- | ------- |
  | —É–±–∞–ª –±—É—Ä—Å—É–ª  |
  |              |         |

  | --mother--   |

  |                       |
  |                       | –≤–µ–ª         | —Ä–µ–ª        |

## Miscellaneous

Copula positioning alters meaning:

| ----------------------- | :------------: | :---------------------------- |
| ==–∏–ª–∞ —Ö—ä–∞—Ç–∏ —Ü–µ== | &nbsp;‚û°Ô∏è&nbsp; | ==–∏–ª–∞ —Ü–µ —Ö—ä–∞—Ç–∏== |
| --[this] is your hat;-- | | --the hat is your;-- |
| <br> |
| ==–¥—É —á–∏–≥—å—É–ª –¥–∞== | &nbsp;‚û°Ô∏è&nbsp; | ==–¥—É –¥–∞ —á–∏–≥—å—É–ª== |
| --I am watching;-- | | --the watching [one] is me.-- |
{.gl}

It also controls emphasis:

- ==—É—Ü—Ü–∏ —Ö—ä–∏–ª–∏ —É–∂–∏–≤ —Ü–µ==  
  --brother [is the one who] _stayed_ home (i.e. stayed, not came, not left);--
- ==—É—Ü—Ü–∏ —Ö—ä–∏–ª–∏ —Ü–µ —É–∂–∏–≤==  
  --brother stayed _home_ (i.e. it is at home where he stayed, not at work);--
- ==—É—Ü—Ü–∏ —Ü–µ —Ö—ä–∏–ª–∏ —É–∂–∏–≤==  
  --_brother_ stayed home (i.e. it is brother who stayed, not sister).--

::: tip
In these last examples the past simple ==—É–∂–∏–≤== [acts as adjective](./verb), whereas the verbal usage would be without the copula (but with the 1st & 2nd person agreement still):

- ::: context
  ==[—É—Ü—Ü–∏]{#1} [—Ö—ä–∏–ª–∏]{#2} [—É–∂–∏–≤]{#3}==  
  --[brother]{#1} [stayed]{#3} [home]{#2};--
  :::
- ==–¥—É —Ö—ä–∏–ª–∏ —É–∂–∏–≤–¥–∞==  
  --I stayed home.--

:::
